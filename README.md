# LEMP stack
a simple LEMP using the *Official* Docker Repository for Alpine Linux, Nginx, MariaDB and phpMyAdmin. PHP uses chialab's version which includes all extensions.

### for start LEMP: 
``docker-compose up -d``
### for stop LEMP:
``docker-compose stop``
### for remove LEMP:
`` docker-compose down``
